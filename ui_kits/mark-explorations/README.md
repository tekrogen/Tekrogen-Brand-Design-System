# Mark explorations

A design canvas presenting seven concept directions for the Tekrogen mark:

| ID         | Name          | Direction                                          |
| ---------- | ------------- | -------------------------------------------------- |
| `v2`       | v2 Baseline   | Reference — existing dragonfly with faceted wings  |
| `quadrant` | Quadrant      | Dragonfly distilled — 4 blades + 1 spine           |
| `eye`      | Compound Eye  | Hex facet cluster — multi-directional awareness    |
| `tetra`    | Tetra         | Isometric tetrahedron, 4 faces = 4 pillars         |
| `diatom`   | Diatom        | Flywheel hub + 4 arms — five-pillar flywheel       |
| `schematic`| Schematic T   | Engineering blueprint monogram                     |
| `codex`    | Codex         | Book spine / git tree hybrid                       |

Each concept is rendered as a `<DCArtboard>` inside a `<DesignCanvas>`. A
floating Tweaks panel lets the reviewer change theme (paper / light / dark),
hero variant (light / dark / mono), wordmark style, tilt offset, and toggle
the lockup + favicon rows.

The **v2 dragonfly** is the production mark; the other six are alternates
kept for reference. If we ever revisit the mark direction, start here.

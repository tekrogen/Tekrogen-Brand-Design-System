# Master lockup sheet

The brand foundations reference document. A long-scroll React app that
covers, in order:

1. **Master** — stacked lockup at production sizes + favicon scale
2. **Rules** — clear-space (1× cap-height) + minimum size table
3. **Dual wordmark system** — Poppins (public) + JetBrains Mono (technical)
4. **Domain split** — one mark, four entities × three intensity levels
5. **Palette** — screen vs print-tuned values, side-by-side comparison
6. **Ghost Pro applications** — header, browser tab, social card

A sticky top toggle lets the reader preview the entire document in either
the Sans (A) or Mono (C) wordmark treatment. **Both are committed** — the
toggle is a visualization aid, not a vote.

Imports `../_shared/lockups-app.jsx`.

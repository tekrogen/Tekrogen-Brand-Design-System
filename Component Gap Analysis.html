Method & scope

Source set: the full default component index at ui.shadcn.com/docs/components. Compared against the kit inventory catalogued in index.html (Components 02) and the preview/ specimens. Already-shipped matches: Button, Card, Input, Avatar, Badge (mono pill), and the Typography scale. Brand-specific surfaces the kit also covers — Dragonfly mark, lockups, OG cards, section heads, eyebrow/meta, glyph + Lucide vocabularies — have no shadcn equivalent and aren't counted here.

Excluded by request: Navigation Menu (handled separately), plus Header and Footer, which already ship in ui_kits/tekrogen-org/ (SiteHeader, Footer). The "Beyond shadcn" group lists kit-native components the brand will want that the shadcn set doesn't define — additive, not part of the 50.
